## Descripción del Proyecto

Desarrollé **TechStore**, una plataforma completa de comercio electrónico especializada en la venta de componentes de PC. 

El proyecto incluye tanto el frontend como el backend, con un sistema de administración robusto y funcionalidades avanzadas de gestión de inventario, cupones de descuento y seguimiento UTM.

---

## Características Principales

### Frontend

**Interfaz moderna** construida con HTML, CSS y JavaScript vanilla para máxima velocidad y control.

- Diseño responsive optimizado para todos los dispositivos
- Sistema de filtros avanzado por categoría, precio y especificaciones
- Búsqueda inteligente con autocompletado y sugerencias
- Carrito de compras con persistencia en localStorage
- Proceso de compra simplificado y seguro

### Backend y Funcionalidades

La arquitectura backend está construida con **Node.js y Express**, ofreciendo una API RESTful completa.

- API RESTful desarrollada con Node.js y Express
- Base de datos para gestión de productos, usuarios y ventas
- Sistema de inventario automático que descuenta stock al realizar compras
- Envío de emails automáticos tras cada compra exitosa
- Gestión de UTMs para tracking de marketing y conversiones

### Panel de Administración

Sistema completo de administración con **autenticación segura** mediante contraseña y captcha.

- Dashboard completo con métricas de ventas y estadísticas en tiempo real
- Gestión de productos con capacidad de agregar, editar y eliminar
- Sistema de categorías dinámico y flexible
- Gestión de cupones de descuento con códigos personalizados
- Análisis de ventas con detección automática de UTMs
- Reportes detallados de ventas y rendimiento

### Sistema de Imágenes Inteligente

Una de las funcionalidades más destacadas es el **sistema de procesamiento de imágenes**:

- Cortado automático a dimensiones óptimas para el sitio
- Conversión automática a formato WebP para mejor rendimiento
- Optimización de peso sin pérdida de calidad visual
- Gestión de stock en tiempo real
- Sistema de notificaciones para administradores

## Desafíos Técnicos Resueltos

### Desafío 1: Gestión Automática de Inventario

**Problema**: Mantener sincronizado el stock entre la base de datos y las ventas en tiempo real sin errores.

**Solución**: Implementé un sistema de transacciones atómicas que actualiza el inventario inmediatamente tras cada compra, con validaciones para evitar sobreventa y mantener la integridad de los datos.

### Desafío 2: Optimización de Imágenes

**Problema**: Las imágenes de productos eran muy pesadas y afectaban significativamente la velocidad de carga del sitio.

**Solución**: Creé un sistema automático que redimensiona, recorta y convierte las imágenes a WebP, reduciendo el peso en un 70% sin pérdida de calidad visual. Esto mejoró drásticamente la experiencia de usuario.

### Desafío 3: Sistema de UTMs

**Problema**: Necesidad de trackear el origen de las ventas para análisis de marketing y optimización de campañas.

**Solución**: Implementé un sistema que captura automáticamente los parámetros UTM y los asocia a cada venta, permitiendo análisis detallados de conversión y ROI de las campañas de marketing.
